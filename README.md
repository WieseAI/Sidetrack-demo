# Sidetrack

A Chrome sidepanel that combines a kanban board with per-task time
tracking. Offline-first, local-only, no accounts, no server.

> This repo is the public showcase of Sidetrack, built autonomously by
> [WieseAI OS](https://wieseai.com) from the brief in
> [`sidetrack-brief.md`](./sidetrack-brief.md). The brief is the source
> of truth for what Sidetrack is. This README is a stub while the
> project is being planned and built — the full landing page lands in
> Phase 5, per the brief's "public repository requirements."
